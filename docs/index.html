Core Architecture

  • Role-Based Access Controls (RBAC)

    Decorators and mixins separating Admin/Librarian views from borrowing Members.

  • Elastic Catalog Search

    Paginated book catalog indexed for rapid queries on titles, authors, and ISBNs.

  • Atomic Borrowing & Returns

    Atomic database transactions ensuring copy quantities and return constraints remain correct.
